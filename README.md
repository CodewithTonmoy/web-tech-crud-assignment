# Web Technology CRUD Assignment

This project demonstrates CRUD (Create, Read, Update, Delete) operations using PHP and MySQL.

## Features
- Add product
- Display products (only displayable items)
- Calculate profit
- Edit product
- Delete product
- AJAX-based search

## Technologies Used
- PHP
- MySQL
- HTML
- JavaScript (AJAX)

## How to Run
1. Install XAMPP
2. Place project inside `htdocs`
3. Create database `product_db`
4. Import table structure
5. Run using `http://localhost/product_crud/`
